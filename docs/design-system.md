# 5️⃣ VISUALS — How does it feel in the hand?

Visuals are **emotion carriers**, not decoration.

### Visual principles

- **Calm > exciting**
- **Space > density**
- **Motion > noise**
- **Earth tones > neon**
- **Human language > icons**

### Map visuals

- **Soft lines** for routes
- **Hand-drawn feel** (slight imperfections)
- **Pins feel placed**, not stamped

### Typography

- **Large, readable**
- **Friendly curves** (Libre Baskerville)
- **Comfortable spacing**

### Micro-interactions

- **Gentle haptic** on save
- **Soft fade-in** of pins
- **Slow zoom** when replaying a journey

### Color emotion

| Element | Color | Feeling |
| --- | --- | --- |
| **Greens** | `#7E9C82` (Sage) | Nature |
| **Sand** | `#F5F5F0` (Mist) | Warmth |
| **Slate** | `#2C4A52` (Deep Forest) | Calm |
| **Accent** | `#D4A373` (Amber) | Discovery |
