# 3️⃣ FLOW — How does someone move through it?

Flow is **storytelling in interaction**.

### Macro Flow (lifecycle)

1.  **Discover app**
2.  **Choose identity (persona)**
3.  **Explore or create**
4.  **Capture moments**
5.  **Organize meaning**
6.  **Decide privacy**
7.  **Share intentionally**
8.  **Return & reflect**

### Micro Flow (pin creation)

```
See → Long press → Capture → Save → Continue
```

-   **No dead ends.**
-   **No forced decisions.**

### Rule

> **Every flow must end in either relief or curiosity.**
